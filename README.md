## 安卓实验3 Intent

**实验内容：**

      本实验通过自定义WebView加载URL来验证隐式Intent的使用。
      实验包含两个应用：
      第一个应用：获取URL地址并启动隐式Intent的调用。
      第二个应用：自定义WebView来加载URL
